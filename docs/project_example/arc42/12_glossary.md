# 12. Glossaire

Termes clés utilisés dans cette architecture et leur définition.

| Terme | Définition |
| :--- | :--- |
| **ADR** | Architecture Decision Record - Fiche documentant une décision structurante. |
| **arc42** | Modèle de structure standardisé pour documenter l'architecture logicielle. |
| **iSAQB** | International Software Architecture Qualification Board. |
| **MCP** | Model Context Protocol - Protocole ouvert permettant aux IA de se connecter à des outils et sources de données. |
| **React Flow** | Bibliothèque React permettant de concevoir des graphes interactifs. |
